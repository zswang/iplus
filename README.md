iplus
-----------

# [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Coverage Status][coverage-image]][coverage-url]

Timestamp based GUID

## 使用方法

### 获取新 ID

```js
var guid = iplus.newGuid();
```

### 获取新 ID，加前缀

```js
var guid = iplus.newGuid('wx');
```
